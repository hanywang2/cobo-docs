---
title: Verify Valid Address
api: "GET /v1/custody/is_valid_address/"
description: Check if a cryptocurrency address is valid.
---


#### Request
<ParamField body="coin" type="String" required>
 Coin code
</ParamField>
<ParamField body="address" type="String" required>
Deposit address(Address|Memo)
</ParamField>

#### Response
<ResponseField name="result" type="bool" required>
address is valid or not
</ResponseField>

<RequestExample>
```python Python
request(
  "GET",
  "/v1/custody/is_valid_address/",
  {
    "coin": "ETH",
    "address": "0x544094588811118b7701cf4a9dea056e775b4b4e",
  },
  api_key, api_secret, host
)
```
