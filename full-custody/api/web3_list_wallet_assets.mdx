---
title: 
api: GET /v1/custody/web3_list_wallet_assets/ 
description: Get Wallet Assets List 
---
#### Request 

<ParamField body="address" type="String" >Address, if it is empty, it will return the assets in all addresses, if it is not empty, it will return the assets in this address</ParamField> 
<ParamField body="chain_code" type="String" >Chain code, if it is empty, it will return all chain_code assets, if it is not empty, it will return the assets in this chain_code</ParamField> 

#### Response 

<ResponseField name="chain_code" type="String" >Chain code</ResponseField> 
<ResponseField name="coin" type="String" >Coin code</ResponseField> 
<ResponseField name="amount" type="String" >Coin/NFT amount</ResponseField> 
<ResponseField name="nft_code" type="String" >NFT Collection Code</ResponseField> 
<ResponseField name="standard" type="String" >Token standard</ResponseField> 
<ResponseField name="contract_address" type="String" >NFT Contract Address</ResponseField>
