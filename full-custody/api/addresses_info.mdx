---
title: Get Addresses Info
api: "GET /v1/custody/addresses_info/"
description: Returns information about the given addresses for a specific coin.
---


#### Request
<ParamField body="coin" type="String" required>Coin code</ParamField>
<ParamField body="address" type="String" required>(Address \| Memo) ,Deposit address Separate by comma more than one source address</ParamField>

#### Response
<ResponseField name="coin" type="String">Coin code</ResponseField>
<ResponseField name="address" type="String">
 Only return the addresses that belong to it correctly, multiple addresses are separated by commas (if there is Memo, separated by vertical bars)
</ResponseField>
