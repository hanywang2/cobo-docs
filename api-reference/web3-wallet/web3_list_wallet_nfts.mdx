---
title: 
api: GET /v1/custody/web3_list_wallet_nfts/ 
description: Obtain the list of NFTs through NFT Code query 
---
#### Request 

<ParamField body="nft_code" type="String" required>NFT Collection Code</ParamField> 
<ParamField body="address" type="String" >Address, if it is empty, it will return the assets in all addresses, if it is not empty, it will return the assets in this address</ParamField> 

#### Response 

<ResponseField name="nft_code" type="String" >NFT Collection Code</ResponseField> 
<ResponseField name="chain_code" type="String" >Chain code</ResponseField> 
<ResponseField name="description" type="String" >Full name of NFT (not unique, changeable, for reference only )</ResponseField> 
<ResponseField name="amount" type="String" >NFT amount</ResponseField> 
<ResponseField name="standard" type="String" >Token Standard</ResponseField> 
<ResponseField name="contract_address" type="String" >NFT Contract Address</ResponseField> 
<ResponseField name="token_id" type="String" >NFT Token Id</ResponseField> 
<RequestExample>
```python Python
request(
    "GET",

    {
```
