---
title: 
api: GET /v1/custody/web3_list_wallet_address/ 
description: Obtain the list of addresses through chain code query 
---
#### Request 

<ParamField body="chain_code" type="String" required>Chain code</ParamField> 
<ParamField body="page_index" type="Int" required>Page index, zero for the first page</ParamField> 
<ParamField body="page_length" type="Int" required>Number of rows per page. (Not less than 1 and not more than 50)</ParamField> 
<ParamField body="sort_flag" type="Int" >Address sorting method: If it is 1, it will be returned in a positive order of address creation time, or else it will be returned in a reverse order of address creation time.</ParamField> 

#### Response 

<ResponseField name="addresses" type="String" >Address</ResponseField> 
<RequestExample>
```python Python
request(
    "GET",
    "/v1/custody/web3_list_wallet_address/",
    {
        "chain_code": "ETH",
        "page_index": 0,
        "page_length": 10,
        "sort_flag": 0
    },
    api_key, api_secret, host
)
```
