---
title: Get Internal Address Info
api: "GET /v1/custody/internal_address_info/"
description: Check Loop Address Details.
---
To check if the address available for Loop transaction.

More info about Loop ï¼š [https://loop.top/](https://loop.top/)

#### Request
<ParamField body="coin" type="String" required>
 Coin code
</ParamField>
<ParamField body="address" type="String" required>
Deposit address
</ParamField>
<ParamField body="memo" type="String">
Memo for specified coins(EOS,XLM,XRP,IOST)
</ParamField>

#### Response
<ResponseField name="coin" type="String">
 Coin code
</ResponseField>
<ResponseField name="address" type="String">
 Deposit address
</ResponseField>
<ResponseField name="is_internal_address" type="Bool">
 If it's Loop address
</ResponseField>
<ResponseField name="internal_org" type="String">
Organization of the address
</ResponseField>
<ResponseField name="internal_wallet" type="String">
Wallet of the address
</ResponseField>
