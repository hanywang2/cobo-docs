---
title: Get Address Info
api: "GET /v1/custody/address_info/"
description: Returns information about a given address for a specified coin.
---


#### Request

<ParamField body="coin" type="String" required>
 Coin code
</ParamField>
<ParamField body="address" type="String" required>
Deposit address(Address|Memo)
</ParamField>

#### Response
<ResponseField name="coin" type="String" required>
 Coin code
</ResponseField>
<ResponseField name="address" type="String" required>
Deposit address(Seperate memo by vertical bar)
</ResponseField>

<RequestExample>
```python Python
request(
  "GET",
  "/v1/custody/address_info/",
  {
    "coin": "ETH",
    "address": "0x544094588811118b7701cf4a9dea056e775b4b4e",
  },
  api_key, api_secret, host
)
```
