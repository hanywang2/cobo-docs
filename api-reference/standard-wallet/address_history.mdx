---
title: Get Address History
api: "GET /CustodyApi/address_history/"
description: Returns a list of addresses associated with the specified coin and custody wallet.
---

#### Request
<ParamField body="coin" type="String" required>Coin code</ParamField>
<ParamField body="page_index" type="Int">Page index, zero for the first page.  </ParamField>
<ParamField body="page_length" type="Int">Number of rows per page. (Not less than 1 and not more than 50)</ParamField>
<ParamField body="sort_flag" type="Int">Address sorting method: If it is 1, it will be returned in a positive order of address creation time, or else it will be returned in a reverse order of address creation time. </ParamField>


#### Response
<ResponseField name="coin" type="String" required>
 Coin code
</ResponseField>
<ResponseField name="address" type="String" required>
Deposit address(Seperate memo by vertical bar)
</ResponseField>

```python Python
request(
```
