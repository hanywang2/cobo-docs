---
title: Change Binding
api: POST /v1/custody/auth/change_binding/ 
description: "This endpoint allows authenticated users to change their binding to a new Prime Broker. The user's `broker_user_id` is passed in the request body, and the new `binder_id` is returned in the JSON response."
---
#### Request 
<ParamField body="binder_id" type="String" required>CoboAuth binding code</ParamField>

#### Response
<ResponseField name="binder_id" type="String" >CoboAuth binding code</ResponseField>
